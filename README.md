# Loan System

## Description
A WinForms-based loan management system that allows customer registration,
loan calculation, and automatic payment schedule generation.

## Features
- Customer management
- Loan calculation with interest
- Automatic payment schedules
- SQL Server database integration

## OOP Concepts Used
- Encapsulation
- Abstraction (abstract base class)
- Inheritance
- Polymorphism
- Interfaces
- Event handling
- LINQ

## Setup
1. Restore database
2. Configure connection string
3. Run the application
